1. The Desktop

The desktop is your main workspace. It displays:

  • Desktop icons — Shortcuts to Files, Terminal, App Center, and Trash
  • Desklets — Floating widgets like clocks and calendars
  • A watermark — Shows "EverestOS - Web operating system"

Right-click the desktop to access options like:

  • Add/remove desklets
  • Change desktop settings (wallpaper, icon visibility)
  • Create new folders and files
  • Open the Looking Glass console

💡 Desktop Icon Positions

Desktop icons can be dragged to custom positions. Your layout is automatically saved to ~/.config/desktop-positions.json.

2. The Panel (Taskbar)

The panel sits at the bottom (or top) of the screen and contains three zones:

ZoneContents
LeftApp Menu button, quick-launch applets
CenterWindow list (open application buttons)
RightSystem tray — clock, date, status indicators

Right-click the panel to access:

  • Panel position (move to top or bottom)
  • Add/remove applets
  • Extension manager
  • System settings

3. App Menu

Click the App Menu button (bottom-left of the panel) to launch applications.

  • Quick Actions — Top row has buttons for Files, Terminal, Settings, and About
  • Search — Type to filter apps by name
  • Categories — Browse apps organized by type (System, Multimedia, Office, etc.)
  • All Applications — See everything in one list

4. Window Management

Application windows behave like a traditional desktop:

  • Drag title bar — Move the window
  • Drag edges/corners — Resize the window
  • Close button (✕) — Close the window
  • Minimize button (−) — Minimize to taskbar
  • Click taskbar entry — Restore or focus window
  • Click any window — Bring it to front

5. File Management

The Files app provides a familiar file browser:

  • Navigate — Click folders to open them, use breadcrumb path bar
  • Home (~/) — Your personal directory
  • Desktop (~/Desktop) — Items shown on the desktop
  • Documents, Music, Videos, images — Organized content folders
  • Trash — Deleted items can be restored or permanently removed

computer:// — Browse the filesystem root. trash:// — View deleted files. Both are accessible from the Files app sidebar.

8. Extensions & Plugins

EverestOS supports two types of plugins that integrate into the desktop shell:

Applets (Panel Widgets)

Small tools embedded in the panel/taskbar. Examples include the main menu, date/time display, and system indicators.

  • Add: Right-click panel → "Add Applets"
  • Configure: Right-click the applet → "Settings"
  • Remove: Right-click the applet → "Remove"

Desklets (Desktop Widgets)

Floating widgets on the desktop surface. They can be dragged to any position and persist across sessions.

  • Add: Right-click desktop → "Add Desklets"
  • Move: Drag the desklet by its header
  • Configure: Right-click → "Settings"
  • Remove: Right-click → "Remove this desklet"

🧩 Plugin Directory

All plugins are stored in ~/Plugins/. Applets in ~/Plugins/applets/ and desklets in ~/Plugins/desklets/. You can also manage them from the Plugin Settings app.

9. Configuration

All user settings are stored as JSON files in ~/.config/:

FileWhat It Controls
theme.jsonActive color theme and icon pack
appearance.jsonPanel opacity, blur, border radius
panel.jsonPanel position (top/bottom)
desktop.jsonDesktop icon visibility (Files, Trash, etc.)
display.jsonUI scaling percentage, night light toggle
startup.jsonApps to auto-launch at boot
extensions.jsonWhich extensions are loaded/removed
menu.jsonApp Menu favorites and settings

📝 Editing Config Manually

You can edit these files directly using the Text Editor or Terminal. Changes take effect after refreshing the session.

10. Debugging (Looking Glass)

Press Alt + F2 to open the Looking Glass — an interactive debug console inspired by Cinnamon's diagnostic tool.

  • View logs — System startup messages, extension errors, and app activity
  • Run JavaScript — Execute code directly in the shell context
  • Inspect extensions — See which plugins are loaded and their state
